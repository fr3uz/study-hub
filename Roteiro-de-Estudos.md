**O que estudiar:**
- [ ] O que é a web (navegador, servidor, requisição/resposta)
- [ ] O que é HTML (HyperText Markup Language)
- [ ] Estrutura básica de um documento HTML (`<html>`, `<head>`, `<body>`)
- [ ] Tags básicas: `<h1>` a `<h6>`, `<p>`, `<a>`, `<img>`, `<ul>`, `<ol>`, `<li>`
- [ ] Atributos (src, href, alt)

**Prática:**
- [ ] Criar uma página HTML com título, parágrafo e imagem
- [ ] Criar lista ordenada e não ordenada
